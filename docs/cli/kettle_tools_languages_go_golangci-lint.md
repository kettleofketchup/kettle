## kettle tools languages go golangci-lint

Install lint tool for Go

### Synopsis

Downloads and installs the latest version of golangci-lint.

```
kettle tools languages go golangci-lint [flags]
```

### Options

```
  -h, --help   help for golangci-lint
```
